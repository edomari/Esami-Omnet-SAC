## Parte 2

- λ = 200 req/s
- Tipo 1: μ=8, costo=1.5$/h
- Tipo 2: μ=16, costo=3$/h  
- SLA: T_r < 0.25s

**Formula M/M/1**
```
T_r = 1/(μ - λ/N) = N/(μN - λ)
```
**Tipo 1 ($\mu$=8)**

```
N/(8N - 200) ≤ 0.25

N/(8N - 200) = 0.25
N = 0.25(8N - 200)
N = 2N - 50
N = 50
```
```
λ/server = 200/50 = 4
T_r = 1/(8-4) = 0.25s
Costo = 50 × 1.5 = $75/h
```

**Tipo 2 ($\mu$=16)**

```
N/(16N - 200) ≤ 0.25
N/(16N - 200) = 0.25
N = 0.25(16N - 200)
N = 4N - 50
3N = 50
N = 16.67 → 17
```
```
λ/server = 200/17 = 11.76
T_r = 1/(16-11.76) = 0.236s
Costo = 17 × 3 = $51/h
```

**Risultati**

| server_type | $N$  | $T_r$     | Costo   |
|------|----|---------|---------|
| tipo1    | 50 | 0.250s  | $75/h   |
| tipo2    | 17 | 0.236s  | $51/h   |

**Più conveniente: Tipo 2** (risparmio $24/h)
