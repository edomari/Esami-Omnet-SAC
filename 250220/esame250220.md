## Parte 2

- λ = 100 req/s
- μ = 10 req/s (per server)
- cv = 3 (lognormal)
- SLA: T_r < 0.25s

```
T_r = (1/μ) · [1 + (ρ/(1-ρ)) · ((1 + cv_s²)/2)]
T_r = 0.1 · [1 + 5ρ/(1-ρ)]
dove ρ = λ/(N·μ) = 100/(10N) = 10/N
```

**Obiettivo**: Trovare $N$ minimo per $T_r \leq 0.25s$

**Imposta $T_r \leq 0.25$:**
```
0.1 · [1 + 5ρ/(1-ρ)] ≤ 0.25

1 + 5ρ/(1-ρ) ≤ 2.5

5ρ/(1-ρ) ≤ 1.5

5ρ ≤ 1.5(1-ρ)

5ρ ≤ 1.5 - 1.5ρ

6.5ρ ≤ 1.5

ρ ≤ 0.231
```

**Ricava N:**
```
10/N ≤ 0.231

N ≥ 10/0.231

N ≥ 43.3

N = 44 server ✓
```

**Verifica N=44**

```
ρ = 10/44 = 0.227

T_r = 0.1 · [1 + 5·0.227/(1-0.227)]
T_r = 0.1 · [1 + 1.135/0.773]
T_r = 0.1 · 2.47
T_r = 0.247s < 0.25s ✓
```

| N  | T_r   |
|----|-------|
| 44 | 0.247s|

**Soluzione: N = 44 server**
