## Parte 2

- λ = 200 req/s
- Tipo 1: μ=10, costo=1.5$/h
- Tipo 2: μ=15, costo=2.5$/h  
- SLA: T_r < 0.25s

**Formula M/M/1**
```
T_r = 1/(μ - λ/N) = N/(μN - λ)
```

**Tipo 1 (μ=10)**

```
N/(10N - 200) ≤ 0.25

N/(10N - 200) = 0.25
N = 0.25(10N - 200)
N = 2.5N - 50
1.5N = 50
N = 33.33 → 34 ✓
```
```
λ/server = 200/34 = 5.88
T_r = 1/(10-5.88) = 0.243s ✓
Costo = 34 × 1.5 = $51/h
```

**Tipo 2 (μ=15)**

```
N/(15N - 200) ≤ 0.25

N/(15N - 200) = 0.25
N = 0.25(15N - 200)
N = 3.75N - 50
2.75N = 50
N = 18.18 → 19 ✓
```
```
λ/server = 200/19 = 10.53
T_r = 1/(15-10.53) = 0.224s ✓
Costo = 19 × 2.5 = $47.5/h
```

**Risultati**

| Tipo | N  | T_r     | Costo   |
|------|----|---------|---------|
| 1    | 34 | 0.243s  | $51.0/h |
| 2    | 19 | 0.224s  | $47.5/h |

**Più conveniente: Tipo 2** (risparmio $3.5/h)
